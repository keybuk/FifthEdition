# Hacking

- Format files using <https://github.com/nicklockwood/SwiftFormat>.
- If using Xcode, install the Source Editor Extension to add "Editor → SwiftFormat → Format File".

## Schema

Parses and generates data conforming to the 5etools schema <https://github.com/TheGiddyLimit/5etools-utils/> 0.16.14, up to the following versions:

 * `schema-template/adventures.json`: 1.10.35
 * `schema-template/bestiary/bestiary.json`: 1.21.62
 * `schema-template/bestiary/index.json`: 1.0.0
 * `schema-template/books.json`: 1.2.20
 * `schema-template/corpus-shared.json`: 1.0.1
 * `schema-template/entry.json`: 1.9.17
 * `schema-template/util.json`: 1.18.6
 * `schema-template/util-edition.json`: 1.0.1
 * `schema-template/util-foundry.json`: 1.1.9
 * `schema-template/util-token.json`: 1.0.3
